# TikTok (国际版) 内容创作指南

## 定位
全球短视频平台，算法驱动，以娱乐、教育、生活方式内容为主。

## 视频规格
- 推荐时长：15-60 秒（最佳），最长 10 分钟
- 竖屏 9:16，分辨率 1080×1920
- 前 3 秒必须抓眼球（视觉冲击 / 悬念 / 强钩子）

## 文案规范
- Caption 上限 2200 字，实际展示前 3 行
- Hashtag：5-10 个（混合 niche + 大众 tag）
- 避免敏感词、品牌名（防限流）

## 内容策略
- 趋势音乐 + 趋势话题 = 更大推流
- 结尾引导：点赞 / 关注 / 评论 / 分享
- 字幕：英文字幕提升完播率 15-20%
- 挑战 (Challenge) 参与可借势流量

## 高效内容类型
- 教程 / How-to（前3秒问题，最后答案）
- 生活 Vlog
- 产品展示 / Unboxing
- 剧情 / 反转
- Duet / Stitch 互动

## 发布节奏
- 建议每天 1-2 条，保持活跃
- 最佳时间（EST）：06:00-10:00 / 19:00-23:00

## 禁忌
- 镜像 / 水印内容（严重限流）
- 版权音乐（商业账号必须用商业音乐库）
- 暴力 / 误导性内容
