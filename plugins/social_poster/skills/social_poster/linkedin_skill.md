# LinkedIn 内容创作指南

## 定位
职场社交网络，受众为职业人士、B2B 决策者、招聘方。

## 内容风格
- **专业、有洞察、有深度**，避免过于口语或娱乐化
- 以"行业观点 / 经验分享 / 案例故事"为主轴
- 以第一人称叙事（"我"/"我们"）更有真实感
- 开头前 2-3 行决定是否展开，必须有吸引力

## 格式规范
- 字数：300-1500 字最佳（上限 3000）
- 段落短：每段 2-3 行，大量换行，方便手机阅读
- Emoji 少而精，不超过 3 个
- Hashtag：3-5 个，放在末尾
- 可用"1. 2. 3."列表，结构清晰

## 高效话题
- 职业经验 / 职场教训
- 行业趋势分析
- 工具推荐 / 效率方法
- 团队文化 / 招聘信息
- 产品 / 案例展示（配数据）

## 发布建议
- 最佳时间：周二至周四 08:00-10:00
- 多用图片 / 文档 / 视频，提升曝光
- 前 60 分钟快速回复评论，触发算法加权

## 禁忌
- 过度推销产品
- 政治话题
- 负面吐槽
- 刷屏式互动群（违反平台规则）
