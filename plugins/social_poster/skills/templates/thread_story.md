# 推文串 / Thread 叙事模板

适用：Twitter/X Thread、LinkedIn 系列帖

---

## 结构框架（5-10 条）

```
第1条（钩子）
→ 强开场：数据 / 反常识 / 问题
→ 预告全文价值："我来分享X个关键点🧵"
→ 控制在 240 字以内

第2-N条（主体）
→ 每条一个观点，独立成章
→ 末尾用"→"或数字引导下一条
→ 可插入图片、数据截图

最后一条（CTA）
→ 总结 + 引导关注 / 转发
→ "如果有帮助，请转发给需要的人 🔁"
```

---

## Twitter Thread 示例

```
1/ 我在 AI 行业工作 5 年，这是我学到最重要的 3 件事 🧵

2/ 第一：工具永远没有思维重要
大家都在学 ChatGPT、Midjourney，
但真正的差距在于你如何提出正确的问题。

3/ 第二：速度比完美更值钱
AI 时代，迭代周期从"月"变成"天"。
等你打磨完，市场已经变了。

4/ 第三：人际关系的护城河无法被 AI 复制
信任、背书、推荐——这些依然是最稀缺的资源。

5/ 总结：
· 培养提问能力
· 快速行动
· 深化人际关系

如果对你有帮助，请转发 🔁
关注我，每周分享 AI 实战心得。
```

## 注意事项
- 使用 `split_text_into_thread` tool 自动分割长文
- 每条推文建议 200-260 字（留余量给序号）
- LinkedIn thread 可以合并为一篇长文发布
