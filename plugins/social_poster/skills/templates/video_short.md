# 短视频文案模板 (Video Short)

适用：抖音、TikTok、Instagram Reels、小红书视频笔记

---

## 结构框架

```
【钩子 Hook — 前3秒】
[痛点问题 / 震惊数据 / 强反差开场]
→ 例："你是不是也有这个问题？"

【价值主体】
[3步解决 / 亲测有效方法 / 故事展示]
→ 保持节奏快，每个要点 5-10 秒

【结尾 CTA】
[引导互动]
→ "关注我，每天分享干货" / "保存这条视频！"
```

---

## Caption 文案公式

```
[一句话总结] + [3个核心关键词]

正文（可选）：
· 要点1
· 要点2
· 要点3

[3-8个 Hashtag]
```

---

## 示例（产品测评）

```
这款防晒霜被低估了！实测3个月总结👇

· 上脸不搓泥、不假白
· 防晒指数 SPF50+ PA++++
· 敏皮也能用，不刺激

#防晒推荐 #平价好物 #护肤日常 #测评
```

---

## 注意事项
- Caption 前 3 行是预览区，关键信息放在前面
- Hashtag 数量：抖音 3-5 个 / TikTok 5-8 个 / Reels 8-12 个
- 视频内字幕和 Caption 内容可以不同，互补效果更好
